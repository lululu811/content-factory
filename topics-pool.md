## 二、选题池管理脚本

```bash
# 查看当前池子
python3 scripts/pool-manager.py --list

# 添加新选题
python3 scripts/pool-manager.py --add \
  --title "光模块 1.6T 演进" \
  --moc "MOC-AI算力" \
  --priority "high" \
  --notes "等英伟达 Rubin 时间表确认后写"

# 标记选题已完成
python3 scripts/pool-manager.py --mark-done \
  --title "光模块 1.6T 演进" \
  --article-slug "optical-module-1.6t-evolution"

# 本周推荐（按 MOC + 时效性 + 优先级）
python3 scripts/pool-manager.py --weekly-pick
```

## 四、选题池使用流程

```
每周日晚：
1. 跑 topic-scorer.py 看本周高分候选
2. 查 Obsidian 库 /query 看库内高频概念
3. 朋友圈 / 微信群收集
4. 加入选题池

每周一晚：
1. 打开选题池（topics-pool.md）
2. 跑 5 个问题过滤（搜索/增量/产能/成长/动机）
3. 选 2-3 个高分题进入本周计划
4. 写入 drafts/selected-{week}.md
```
