## 表达骨架

### 开头

> 全球每 10 个新建数据中心,4 个在东南亚——而马来西亚柔佛州承接了 70%+。

### 中段 1:选址 5 维度

- 土地:单价 / 可用面积 / 政策
- 电力:成本 / 稳定性 / 绿色占比
- 网络:海底光缆 / 国际带宽 / 时延
- 政策:数据合规 / 税收优惠 / 政府支持
- 人才:工程师 / 运维人员 / 语言

### 中段 2:马来西亚柔佛州

- 紧邻新加坡 1.5 km(新柔长堤)
- 土地充足 + 成本低(新加坡 1/5)
- 电力充足 + 绿色占比提升中
- 中国万国数据 + 碧桂园合作

### 中段 3:新加坡高端

- 土地稀缺(728 km²)
- 强制绿色认证(可再生能源 100%)
- 2027 起 PUE < 1.3 强制
- 承接"高端 + 绿色"差异化

### 中段 4:越南/泰国配套

- 越南胡志明 + 越南北宁 + 泰国曼谷 = 3 大封测集群
- 服务数据中心硬件需求
- 中国 + 东盟 = 完整 AI 产业链

### 中段 5:投资配置

- **本土**:YTL / 吉宝数据中心 REIT
- **跨境**:万国数据(GDS.US) + 碧桂园服务(6098.HK)
- **上游**:Amkor / ASE / Intel 越南(封测)
- **下游**:微软 / 谷歌 / AWS(超大规模云)

### 结尾

> 东盟数据中心是中国 AI 算力的"后花园"——不是替代,是延伸。
