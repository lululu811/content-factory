## 核心内容

**中国核电 2025 年核心数据**:

| 指标 | 数值 |
|---|---|
| 营业收入 | **820.75 亿(+6.22%)** |
| 归母净利润 | **93.04 亿(+6.00%)** |
| 在运机组 | 25 台 |
| 利用小时 | 7800+ |
| ROE | ~10% |

**中国广核 2025 年核心数据**:

| 指标 | 数值 |
|---|---|
| 营业收入 | **756.97 亿(-4.11%)** |
| 归母净利润 | **97.65 亿(-9.9%)** |
| 在运机组 | **28 台 5606 万千瓦** |
| 平均利用小时 | **7767** |
| 分红比例 | 45%(中国核电 35%) |

**双寡头结构**:
- 中国只有 2 家核电运营牌照(中核集团 + 中广核集团)
- 民营/外资进不来,新进入者基本为零
- 2026 核电保底机制:按长期合同价收购(0.43 元/千瓦时),不再参与现货"卷价格"

**核心洞察**:
- 利用小时 7800 全国最高(火电 4500、水电 3500、风光 1088-2000)
- **全天 89% 时间满负荷运行** = 最匹配的清洁能源
- 保底机制落地后,核电估值修复空间 20-30%(PE 从 12-15 修复到 15-18)
- 中国广核营收/净利双降,但**分红比例 45%**(高于中国核电 35%)
