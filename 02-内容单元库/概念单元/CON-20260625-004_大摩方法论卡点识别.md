## 核心内容

**大摩方法论** —— AI 算力供应链卡点识别 4 步:

1. **拆解 8 层级**:L1 下游 → L2 系统 → L3 GPU → L4 服务器 → L5 HBM → L6 制造 → L7 资源 → L8 电力
2. **标定扩产难度**:每个层级评估技术 / 资本 / 政策 / 时间约束
3. **识别隐性卡点**:L7-L8 比 L3 更卡(电力建设周期 3-5 年 vs GPU 1-2 年)
4. **匹配投资时点**:长期(L8)/ 中期(L7)/ 短期(L3-L6 概念)

**核心原则**:
- **显性瓶颈被市场充分定价**(L3 GPU)
- **隐性瓶颈被市场低估**(L7-L8 资源端)
- **信息不对称收益更大**(资源端需要深度调研)
