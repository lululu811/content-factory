# 合规话术对照表

> 写作时必查。🟢🟡🔴 = 红线严重度（🔴 = 绝对不能写）。

---

| ❌ 不要写 | 风险 | ✅ 要写 |
|---|---|---|
| "买入 XX，目标价 100 元" | 🔴 极高 | "值得关注 / 合理估值区间 XX-XX" |
| "独家内幕 / 内部消息" | 🔴 极高 | "基于公开公告 / 财报分析" |
| "保证收益 / 包赚 / 稳赚" | 🔴 极高 | "投资有风险，决策需谨慎" |
| "我预测 X 会涨 50%" | 🟡 高 | "存在上行可能性，需看 XX 信号" |
| "必须配置 / 不能错过" | 🟡 高 | "值得跟踪 / 可纳入观察池" |
| "强烈看好 / 强烈推荐" | 🟡 中 | "值得关注（个人观点）" |
| "近期会突破 / 见顶" | 🟡 中 | "关键信号出现时需重新评估" |
| "推荐仓位 30%" | 🟡 中 | "如果关注这个方向，XX 占比 X%" |
| "目标价 50-60 元" | 🟡 中 | "根据 XX 模型测算区间 50-60 元" |
| "未来 1 年值得持有" | 🟢 低 | "持有期内跟踪 XX 指标" |
| "我判断准确率 X%" | 🟢 低 | "公开追踪战绩表，见 [链接]" |

## 高风险词的替换原则

### 动词类

| 高风险 | 低风险替身 |
|---|---|
| 买入 / 卖出 | 关注 / 跟踪 / 纳入观察 |
| 加仓 / 减仓 | 提高权重 / 降低权重（且需说明条件） |
| 抄底 / 逃顶 | 关注右侧信号 / 等待企稳 |
| 锁仓 / 持有 | 跟踪 / 持续观察 |

### 名词类

| 高风险 | 低风险替身 |
|---|---|
| 目标价 | 合理估值区间 / 测算区间 |
| 必涨 / 必跌 | 概率较高 / 概率较低 |
| 内幕 | 基于公开信息 / 已披露公告 |
| 独家 | 个人整理 / 综合多源 |

### 程度副词类

| 高风险 | 低风险替身 |
|---|---|
| 强烈 / 必须 / 一定 | 值得关注 / 个人倾向 |
| 稳赚 / 包赚 | 存在这种可能 |
| 暴涨 / 暴跌 | 大幅波动 / 明显变化 |

## 自检流程（每篇文章）

1. Ctrl+F 搜索文章是否含以下高风险词：
   - 买入 / 卖出 / 加仓 / 减仓
   - 目标价 / 必涨 / 必跌
   - 内幕 / 独家 / 保证
   - 推荐 / 强烈 / 必须

2. 如果命中：
   - 🔴 红线词 → 整段重写
   - 🟡 黄线词 → 改为对应替身
   - 🟢 绿线词 → 视情况保留

3. 完成后让另一个人（or AI）通读一遍找漏洞
