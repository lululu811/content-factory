## 14 项必查

### A. 内容硬约束（v2 文章标准，缺一项不通过）

- [ ] **1. 标题 ≤ 30 字 + 含反共识钩子**
- [ ] **2. 候选公司 ≥ 20 个,5 分类齐全**（Controls ≥ 2 / Supplies ≥ 4 / Benefits ≥ 4 / Weak ≥ 3 / Story ≥ 2）
- [ ] **3. 9.4 / 9.5 两类必须有具体公司 + 股票代码**（不能用 `...` 或类别描述占位；见 post-template-v2.md 第 8 节硬约束）
- [ ] **4. Top 7 完整 5 要素**（卡住的环节 / 产业链位置 / 排序原因 / 证据 / 主要风险）
- [ ] **5. 反共识方向 ≥ 3 类**（必须包括"排名较低的热门方向"并解释为什么）
- [ ] **6. 配图 ≥ 5 张**（matplotlib 数据图,深色背景 `#0f1419`,不用 emoji 字符）
- [ ] **7. 数据时效：2026 H1 最新**（每条数据标日期 + 来源,如"Mysteel 2026/6/12"）

### B. 合规与证据（监管红线，缺一项直接翻车）

- [ ] **8. 免责声明（中版本）** — 文章末尾 + 提到具体公司处标注"研究案例 / 不构成推荐"
- [ ] **9. 证据等级 🟢🟡🔴 齐全** — 核心判断每条都标等级；强结论必须有 🟢 证据
- [ ] **10. 证据来源可查** — 每条强结论有可查来源（链接 / 截图 / 数据库 / 公告 ID）
- [ ] **11. 高风险词 0 处误用** — "目标价 / 推荐 / 保证 / 加仓 / 稳赚 / 必涨 / 必跌"等只能出现在免责声明反向表达中
- [ ] **12. 短期判断注明时间窗口** — 涉及"业绩兑现 / 订单加速 / 政策落地"等判断时,必须标"6 个月内 / 12 个月内 / 3-6 个月窗口"
- [ ] **13. 升降级信号齐全** — 每篇 v2 标准文章必须有 ≥ 3 条升级信号 + ≥ 3 条降级信号（用 `### 升级信号` / `### 降级信号` 段落锚点,见 tracking-record.py 提取规则）
- [ ] **14. 跟踪记录已写入 tracking/predictions/{slug}.json** — 发布后立刻 `python3 scripts/tracking-record.py add <slug>`,验证日 = 发布日 + 180 天

### C. 访谈类深度文专用（仅对带 `interview:` frontmatter 的文章启用）

- [ ] **15. 引用块 + 中英对照标识**（v3 模板硬约束,仅访谈 / 闭门会 / 高管对话类启用）
  - 引用块 **≥ 10 个** / 篇,分布：一、为什么写这个 ≥ 1 / 四-六、每个卡点 ≥ 2 / 七、5 分类 Controls + Supplies 各 ≥ 1 / 八、Top 7 每个 ≥ 1 / 九、反共识 ≥ 1 / 十一、风险信号 ≥ 1
  - 中英对照标识 **EN ≥ 5 / CN ≥ 5,EN == CN**（用 🇺🇸 / 🇨🇳 emoji + **[EN]** / **[CN]** 标签 + *" "* / 「」 不同引用符号三重对比）
  - 每条引用必须带：**章节名（emoji + 中文标题）** + **行号** + **英文原话（保留口误）** + **中文翻译（修正口误）** + **证据等级**
  - 详见 `templates/post-template-interview.md` 附录 A

## 关键场景加重检查

### 涉及具体标的时
- [ ] 是否有明确 "研究案例 / 不构成推荐" 字样
- [ ] 是否有具体估值数据（目标价 / 仓位建议）→ 删除或改为 "测算区间"
- [ ] 是否有具体买卖时点建议 → 删除
- [ ] 是否引用了非公开信息 → 检查

### 涉及"战绩 / 准确率"时
- [ ] 是否基于公开追踪表（tracking/predictions/）
- [ ] 是否有完整时间窗口说明
- [ ] 是否避免了 "跟着我们买" 等措辞

### 涉及行业 / 政策预测时
- [ ] 是否标注了时间窗口
- [ ] 是否给了 "升降级信号"（什么情况预测需要修正）
- [ ] 是否避免了 "必将 / 一定" 等绝对化措辞

## 检查方式（推荐工具链）

| 工具 | 用途 |
|---|---|
| **本地 Checklist**（本文档） | 写作时勾选 |
| **AI 二次审稿** | 复制全文 + "请按 templates/compliance/checklist.md 14 项逐项检查"（见 scripts/prompt-pack.md 第 5 节） |
| **人工通读** | 关键文章（付费 / 涉及具体标的）必须人工通读 |
| **存档** | 已发布文章存到 archives/,便于回溯；跟踪记录存到 tracking/predictions/,季度回看 |
