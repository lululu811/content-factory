# 合规红线速查卡

> 贴在工作台 / 显示器旁边。每次写作前看一遍。

---

```
🚨 合规红线 5 条 🚨

1. ❌ "目标价" → ✅ "合理估值区间"
2. ❌ "买入/卖出" → ✅ "值得关注"
3. ❌ "承诺收益" → ✅ "投资有风险"
4. ❌ "内幕消息" → ✅ "基于公开信息"
5. ❌ "独家准确率" → ✅ "公开追踪战绩"

任何一条踩了，整篇文章作废。
```

---

## 30 秒自检（写完一篇后）

盯着文章问自己：

1. 这篇能**截图传播**吗？（能 → 加免责）
2. 这篇能**被引用为投资依据**吗？（能 → 加免责 + 改措辞）
3. 这篇有**具体标的**吗？（有 → 加 "研究案例"）
4. 这篇有**具体目标价**吗？（有 → 改为 "估值区间"）
5. 这篇有**承诺性措辞**吗？（有 → 删除）
6. 这篇有**证据等级标注**吗？（没有 → 加）
7. 这篇有**时间窗口声明**吗？（没有 → 加）

7 个问题，3 分钟检查 = 90% 风险规避。

---

## 一句话口诀（必背）

> **强证据 + 弱判断 + 免责声明 = 合规**
> **弱证据 + 强判断 = 翻车**
