# 账号定位文案模板

> 选一个版本作为公众号简介。建议先用版本 A 试 1 个月，看调性是否合适再换。

---

## 版本 A：稳健派（推荐新手起步）

```
本号专注于 [投资方法论 / 产业链研究 / 公开信息分析]。

我们用 AI 工具武装自己，把付费研报、视频内容、公开公告
转化为可消化的研究素材，所有判断基于公开可验证信息。

每周 X 篇深度文，主攻 [N 个] 垂直方向。

⚠️ 本号所有内容仅为研究方法分享，不构成投资建议，
   投资有风险，决策需谨慎。
```

## 版本 B：技术派（推荐技术背景用户）

```
[人名] + AI 副驾的投研实验室。

工具链：
• bibigpt（音视频素材）

• mmx（视觉生产）

每周 X 篇深度文，主攻 [方向]。

⚠️ 研究方法分享，不构成投资建议。
```

## 版本 C：方法论派（最安全）

```
不荐股、不喊单、不承诺收益。

只做三件事：
1. 用 [N 种] 研究方法拆解产业链
2. 用证据等级标注每条判断的可信度
3. 公开追踪我们的预测对错（见战绩表）

⚠️ 投资有风险，决策需谨慎。
```

---

## 使用建议

- 简介字数控制在 200 字以内（公众号简介上限）
- 必须含 "不构成投资建议" 字样
- 联系方式留 1 个（邮箱或微信），方便读者反馈
- 不放个人照片 / 二维码（减少被截图传播风险）
- 不放 "目标读者" 描述（避免被认定为定向投顾）
