# 特殊场景合规

> 容易踩坑的 3 个特殊场景：战绩表 / 非共识 / 引用他人。

---

## 场景 1：战绩表（预测追踪）

### 合规写法

```
✅ 「我们公开追踪自己的判断对错，2026 H1 整体准确率约 78%」
   （事实陈述，可信可查）

✅ 「以下是我们 2026 H1 公开判断与实际结果的对照表，
    完整记录见 [链接]」
   （透明、可验证）
```

### 违规写法

```
❌ 「我们准确率 78%，跟着我们买准没错」
   （虚假宣传 + 荐股）

❌ 「我们预测准确，错过了会后悔」
   （误导 + 软性荐股）
```

### 战绩表合规要求

- 公开完整记录（所有判断，包括错的）
- 时间窗口清晰（如 "2026 H1"，不要笼统说 "我们一直"）
- 数据来源可查
- 不暗示 "跟随" 或 "购买"
- 独立第三方可验证

## 场景 2：非共识判断

### 合规写法

```
✅ 「与市场主流观点不同，我认为 XX（基于公开数据）」
   （个人观点表达）

✅ 「市场主流认为 XX，我的判断是 YY，理由是 ZZ」
   （有论据、有边界）

✅ 「这个判断如果错了，降级信号是 XX」
   （可证伪 → 增加可信度）
```

### 违规写法

```
❌ 「市场错了，我才是对的」
   （傲慢 + 误导）

❌ 「这个标的被严重低估，必定反转」
   （绝对化 + 无证据）

❌ 「反共识的机会不多了，现在不上车来不及」
   （软性荐股 + 制造焦虑）
```

### 非共识判断的合规技巧

1. **明确标 "与主流不同"** —— 不是 "我比市场聪明"
2. **给出判断的时间窗口** —— 不是 "永远正确"
3. **明确 "降级信号"** —— 错了会修正
4. **承认 "可能错"** —— 个人观点
5. **不用情绪化措辞** —— 不是 "市场疯了"

## 场景 3：引用他人观点

### 合规写法

```
✅ 「某券商研报认为 XX（来源：[链接]）」
   （引用 + 标注）

✅ 「多位分析师指出 XX 趋势（综合 [N] 家研报）」
   （综合标注）

✅ 「在 [某会议] 上，[某专家] 分享了 XX 观点」
   （事件标注）
```

### 违规写法

```
❌ 大段复制他人观点不标注
   （抄袭 + 侵权）

❌ 「某券商指出 XX」但没有给出可查的来源
   （虚假引用）

❌ 「据内部消息」 / 「据某不愿具名人士」
   （来源不可信 + 可能违规）
```

### 引用规则

1. **必须可查**（链接 / 截图 / 文档 ID）
2. **必须标注来源**（即使是付费星球）
3. **不能大段复制**（> 100 字需特别小心版权）
4. **隐去个人姓名**（除非公开大 V）
5. **隐去星球名 / 群名**（合规边界）
6. **标注引用类型**（如 "L2 中证据"）

## 其他高频踩坑点

### "独家" 措辞

| ❌ 错 | ✅ 对 |
|---|---|
| "独家内幕" | "基于公开公告分析" |
| "独家模型" | "个人整理 / 综合多源" |
| "独家数据" | "公开数据二次加工" |

### "内幕" 措辞

| ❌ 错 | ✅ 对 |
|---|---|
| "据内部人士" | "据公开披露" |
| "圈子消息" | "行业讨论"（且标 L3） |
| "先于市场知道" | "较早注意到" |

### "保证 / 承诺" 措辞

| ❌ 错 | ✅ 对 |
|---|---|
| "稳赚" | "存在这种可能" |
| "包涨" | "概率较高（个人判断）" |
| "保证收益" | "投资有风险，决策需谨慎" |
| "零风险" | "任何投资都有风险" |
