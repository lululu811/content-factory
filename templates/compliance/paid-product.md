# 付费产品合规（M7+ 启用）

> 当你开始做付费内容 / 服务时启用。开始付费前**强烈建议咨询专业律师**。

---

## 一、服务边界（核心）

### 你提供什么

- ✅ 研究框架 / 方法论
- ✅ 信息加工 / 数据整理
- ✅ 参考案例 / 历史复盘
- ✅ 公开信息分析

### 你**不**提供什么

- ❌ 具体投资建议
- ❌ 目标价 / 估值建议
- ❌ 仓位建议
- ❌ 择时建议
- ❌ 代客理财 / 收益分成
- ❌ 个性化投资咨询

---

## 二、服务协议必备条款

```markdown
# 服务协议

## 第一条 · 服务性质

本服务为「研究方法分享 + 信息加工服务」。

服务内容包括但不限于：
- 研究框架、方法论的整理与分享
- 公开信息的二次加工与解读
- 行业 / 产业链数据整理
- 历史案例分析

服务**不**包括：
- 具体投资建议
- 目标价、估值、仓位建议
- 择时建议
- 代客理财、收益分成
- 任何形式的 "荐股" 服务

## 第二条 · 投资风险

1. 投资有风险，过往业绩不代表未来表现。
2. 用户应根据自身情况独立判断，自担投资风险。
3. 本服务内容不应作为用户投资决策的唯一依据。

## 第三条 · 用户责任

1. 用户应自行核实服务中提供的所有信息。
2. 用户应自行评估自身风险承受能力。
3. 用户应自行做出投资决策，自担后果。

## 第四条 · 免责声明

1. 本服务基于公开信息分析，不保证准确性、完整性、及时性。
2. 服务中提及的任何标的均为研究案例，不代表买入/卖出建议。
3. 作者不对任何因使用本服务而产生的损失负责。

## 第五条 · 知识产权

1. 服务内容的版权归作者所有。
2. 用户可在个人范围内使用，不得用于商业用途。
3. 未经作者书面许可，不得复制、传播、转让。
```

## 三、合规备案（重要）

### 必须咨询律师的场景

- 开始收取年费 / 月费
- 提供"一对一咨询"（可能触及投资咨询牌照）
- 涉及"跟单 / 跟投"（可能触及代客理财）
- 涉及"收益分成"（明确违规）

### 可能不需要牌照的场景

- 公开研究方法分享（带免责声明）
- 信息加工服务（不带个性化建议）
- 数据 / 工具型产品（如 API / 数据库）

### 不确定时

- **付费产品上线前必须咨询专业律师**
- 不同地区法规不同，不要假设
- 律师费是一次性投入，换来的是长期合规

## 四、付费产品的合规经营要点

### 营销合规

| ❌ 不要 | ✅ 要 |
|---|---|
| "跟着我们买，年化 X%" | "我们的研究方法 / 信息加工服务" |
| "VIP 群里有独家标的" | "VIP 群提供更深度的研究框架" |
| "限时优惠，机不可失" | "限时优惠"（不提及紧迫性收益） |
| "实现财富自由" | "建立自己的研究体系" |

### 内容合规

- 每篇付费内容仍要含免责声明
- 付费内容同样不能写目标价 / 荐股
- 付费内容证据等级标注更严（因为读者付费期待更高）
- 涉及具体标的时，必须明确 "研究案例" 标注

### 退款 / 投诉合规

- 明确退款政策（建议 7 天无理由）
- 提供读者反馈渠道
- 投诉处理流程要留痕
- 涉及具体投资损失的投诉 → 立即咨询律师

## 五、付费产品的法律文件清单

上线前必备：

- [ ] 服务协议（用户注册时勾选）
- [ ] 隐私政策（涉及用户信息）
- [ ] 免责声明（每篇付费内容末尾）
- [ ] 退款政策（写在服务协议里）
- [ ] 用户协议（年龄 / 地域限制 / 风险声明）

## 六、一句话总结

> **付费产品的本质是"卖方法论 + 卖时间"，不是"卖投资建议"。**
> **边界清晰 = 长期安全。**
> **越界 = 短期收入 vs 长期风险。**
