# Outline · liquid-cooling · AI 数据中心液冷深度拆解

## 主问题(QST)

**AI 数据中心液冷是 2026 年最被低估的"算力基础设施"机会——它凭什么成为继电力、稀土、光模块之后的第 4 大卡点?**

## 主观点(OPI)

**液冷不是散热技术,是 AI 算力上限的瓶颈。** 当芯片功耗突破 2000W、单机柜功率突破 225kW,液冷从"选配"变成"必配"。2026 渗透率从 14% → 33% → 40%,正是关键拐点年。国产液冷产业链(英维克 + 高澜 + 浪潮 + 中科曙光 + 曙光数创 + 申菱 + 永和 + 同飞 + 远东 + 银轮)将充分享受 86 亿美元市场规模。

## 章节设计

### 第 1 章:液冷不是散热,是算力上限的瓶颈
- 市场误区:液冷 = 数据中心散热技术
- 反共识:液冷 = 决定 AI 算力上限的硬约束
- 关键数据:Rubin GPU 2000W / Vera Rubin NVL 72 225kW 机柜 / 风冷上限 800W

### 第 2 章:3 大液冷路线对比(冷板 vs 浸没 vs 喷淋)
- 冷板式液冷(主流 70%):GB200/GB300/Rubin 都用
- 浸没式液冷(28%):超算 + 智算,效率最高
- 喷淋式液冷(2%):新华三 + 贵安东数西算

### 第 3 章:8 大龙头公司拆解(冷板 / 浸没 / 介质 / 设备链)
- Controls(卡稀缺层):英维克 + 高澜股份 + 曙光数创
- Supplies(服务稀缺层):浪潮信息 + 中科曙光 + 申菱环境 + 远东股份 + 同飞股份
- Benefits(受益趋势):工业富联 + 永和股份 + 银轮股份 + 中航光电 + 东阳光
- Weak(控制力弱):海鸥股份 + 飞荣达 + 大元泵业
- Story(避开):蹭概念的小票

### 第 4 章:2026 渗透率拐点 + 800V HVDC 孪生机会
- 渗透率时间轴:14% → 33% → 40% → 80%
- 800V HVDC:液冷机的孪生兄弟(660kW → 1.2MW)
- 单机柜密度:20kW → 50kW → 100kW → 225kW

### 第 5 章:5 张图 + 反共识 + 操作策略
- 5 张 matplotlib 图
- 反共识方向:氟化液国产替代 / 浸没式被低估 / 3M 退出催化的隐形机会
- 操作策略:按资金量匹配
