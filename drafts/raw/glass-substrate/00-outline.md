## 核心论点

**玻璃基板 = AI 芯片先进封装的下一代底座,2026 商业化元年**

反共识角度:
- 🟢 **不是"概念股"蹭题材** —— 玻璃 vs 有机(ABF)**不是替代,是搭配共存**(RDL + TGV/Cu + ABF 三层结构)
- 🟢 **设备先于材料放量** —— 早周期盯设备(帝尔/德龙/海目星),晚周期看载板(沃格/京东方)
- 🟢 **TGV 是核心瓶颈** —— 玻璃原片 90% 在康宁/肖特/旭硝子手中,激光打孔+填孔电镀才是国产替代真正窗口

- **候选公司**:沃格光电 / 京东方A / 兴森科技 / 蓝思科技 / 彩虹股份

### 4. 设备(早周期机会)
- 激光打孔(帝尔/德龙/海目星/凯格精机)

- **候选公司**:帝尔激光 / 德龙激光 / 大族激光 / 海目星 / 凯格精机 / 东威科技

### 5. 药水+封测(配套环节)

- **候选公司**:天承科技 / 长电科技 / 通富微电 / 华兴源创 / 三孚新科
