## 第 0 部分：核心论点

**电力板块被市场严重低估**，原因：
1. **认知偏差**：散户只看"夏炒电"季节性规律，忽略电力板块的三重属性（周期性 + 强政策性 + 公用事业性）
2. **AI 算力需求**：2025 算力中心用电 1700 亿度（占全社会 1.6%）→ 2030 预计 8000 亿度（占 6%）= **5 年 4.7 倍增长**
3. **容量电价改革**：2024 起火电"两部制电价" + 2026/3 辽宁核电保底机制 = **公用事业估值重估**

**核心判断**：
- 长江电力是水电"印钞机"（净利率 40%）
- 华能国际是火电"打工人"（净利率 8.5%）
- 中国核电 + 中国广核 = 核电双寡头（牌照壁垒）
- 风光运营商 = "我命由天不由我"（利用小时数下滑 + 电价下行）

## 第 1 部分：5 个潜在方向

### 方向 A · 行业逻辑（为什么"夏炒电"这次不一样）

**核心判断**：今年可能打破季节性调整规律 —— 高温（最高用电负荷 16 亿千瓦，+9000 万千瓦）+ 中报业绩预增 + 算电协同入政府工作报告 = 多重 Buff 叠加

**反共识**：市场把"夏炒电"当季节规律，但**电力板块本质是防御性公用事业 + 周期 + 政策三重属性**，过去 3 年估值杀到 PB 1 倍附近，已经到底部

**候选公司**：五大六小阵营（国家能源集团 / 华能 / 大唐 / 华电 / 国家电投 / 三峡 / 中广核 / 华润 / 国投 / 中节能 / 中核）旗下 72 家电力运营商

### 方向 B · 水电（印钞机赛道）

**核心判断**：长江电力断层第一（占全国水电 16%），净利率 40% 远超火电龙头华能国际（8.5%）；雅鲁藏布江工程总投资 1.2 万亿、装机 6000-7000 万千瓦（2030 首台发电），东方电气 + 哈尔滨电气双寡头

**反共识**：水电看着像周期股（来水波动），实际是**最强防御资产** —— 折旧到期后利润进一步释放（大坝实际寿命 80-100 年）

**候选公司**：长江电力（600900） / 桂冠电力 / 国投电力 / 华能水电 / 川投能源 + 东方电气 / 哈尔滨电气（设备）

### 方向 C · 火电（重估窗口）

**核心判断**：两部制电价改革 + 容量电价落地 + 煤价上行（天津港 Q5500 涨到 860 元/吨，+25%）→ 火电从暴涨暴跌的周期股**逐步转向现金流稳定的公用事业**

**反共识**：市场认为火电 = "周期股" = 看煤价吃饭，但实际上**2024 容量电价政策已经把它从"看煤价吃饭"变成"看发电能力吃饭"**

**候选公司**：华能国际（600011） / 华电国际（600027） / 国电电力（600795） / 大唐发电（601991） / 京能电力（600578） / 建投能源（000600） / 皖能电力（000543） / 申能股份（600642）

### 方向 D · 核电（双寡头 + 牌照壁垒）

**核心判断**：中国核电（601985）+ 中国广核（003816）双寡头，2024-2025 连续 4 年每年核准 10+ 台新机组；中国 2030 核电装机超 1.1 亿千瓦超美成第一，2040 装机 2 亿千瓦（占发电量 10%）

**反共识**：核电装机占比最小（仅 2%）但发电量占比 4.6%（利用小时数 7800，全行业第一）；2026/3 辽宁核电保底机制（70% 电量保底价）= 估值重估

**候选公司**：中国核电（601985） / 中国广核（003816） + 设备链：东方电气 / 上海电气 / 哈电集团

### 方向 E · 风光（少熬鹰阶段）

**核心判断**：风光装机占比接近 50% 但发电量只占 22%（利用小时数下滑 + 消纳难题）；三峡能源 + 龙源电力 2025 营收净利双降；**未来核心看点不再是装机增速，而是算电协同 + 储能配套 + 优质区域项目卡位**

**反共识**：风光过去是"成长股"（双碳概念），现在 PB 跌到 0.9-1x，**成长滤镜消失后，谁能绑定算力园区 + 提供稳定绿电，谁才走成长逻辑**

**候选公司**：三峡能源（600905） / 龙源电力（601105） / 华电新能 / 中闽能源 / 江苏新能 / 节能风电 / 林洋能源 / 金风科技 / 阳光电源（储能）

## 第 3 部分：写作章节设计

### 章节 1：行业底层逻辑 —— 电力不是周期股，是三重属性叠加
- 三重属性：周期性 + 强政策性 + 公用事业性
- 历史复盘：2021-2025 行情节奏（3-4 月启动、5 月主升、6-7 月调整）
- 今年不一样：高温 + 中报预增 + 算电协同写入政府工作报告

### 章节 2：四大赛道 —— 4 种完全不同的生意逻辑
- 水电：印钞机（长江电力净利率 40%）
- 火电：转型公用事业（容量电价）
- 核电：双寡头 + 牌照壁垒
- 风光：少熬鹰阶段（绑定算力园区的才有成长）

### 章节 3：核心龙头 —— 7 个标的 5 要素分析
- 长江电力 / 华能国际 / 国电电力 / 中国核电 / 中国广核 / 三峡能源 / 龙源电力

### 章节 4：历史复盘 —— 电价下行 3 年 + 估值分化
- 全国长协电价从 4 毛多降到 3 毛 5 左右
- 风光 PB 从 3.8x 跌到 1x
- 水电 PB 从 2x 涨到 3.5x
- 火电核电 PB 长期 1-1.5x

### 章节 5：拐点机会 —— 算电协同 + 容量电价 + 高温
- 需求端：算力中心用电 2025 年 1700 亿度 → 2030 年 8000 亿度
- 价格端：煤价反弹 + 核电保底机制 = 电价止跌
- 政策端：算电协同写入政府工作报告

### 章节 6：风险信号 + 反共识
- 风险：煤价再次下行 + 电价改革推进慢 + 高温低于预期
- 升级信号：长江电力中报净利 +10% / 华能国际容量电价收入占比 > 30%
- 降级信号：长江电力来水严重偏枯 / 风光利用小时数再降 10%
- 反共识：算电协同不是"概念炒作"，是真金白银的增量需求
