## 核心论点

**康宁 6/24 GlassBridge 把玻璃基板升级为"光互连载板",CPO 1300 亿市场量产拐点**

反共识角度:
- 🟢 **玻璃基板从"封装底座"到"光互连载板"** —— 不是渐进式改良,是对光互连底层架构的重构
- 🟢 **玻璃桥不是"取代 FAU",是"耦合方式改变"** —— 飞书日报 6/25 明确"落地仍需时间"
- 🟢 **CPO 量产比预期慢,但 NPO 可能加速** —— 中信建投 6/15:"推迟至 2028-2029",但 NPO 是通用方案

## 5 章结构

### 1. 玻璃基板回顾(20%)
- 接续 glass-substrate:玻璃 vs ABF 不是替代,是搭配共存
- 6/24 康宁发布 GlassBridge = 玻璃基板故事的"第二幕"
- 玻璃基板从"封装底座"升级为"光+电"双重功能载体

### 2. 玻璃桥(GlassBridge)技术深度(30%)
- 6/24 韩国首尔 AI 数据中心光通信与互连技术大会
- 技术原理:"光纤 → 玻璃波导 → PIC"用半导体级离子交换工艺
- O 波段损耗 <0.1dB/cm
- 首款产品:30 微米间距 + <2dB 耦合损耗
- 与 GlobalFoundries 深度绑定

### 3. 玻璃桥 vs FAU 对比(15%)
- 传统方案:光纤 → FAU 机械对准 → PIC(精度要求高 + 良率难控)
- 玻璃桥:光纤 → 玻璃波导 → PIC(半导体工艺)
- 解决"尺寸鸿沟":片上光波导数百纳米 vs 光纤数微米
- 与 Meta / 英伟达 / 亚马逊签署数十亿美元长期供货协议

### 4. CPO 1300 亿光互连新空间(20%)
- 花旗预测:2028 全球光互连市场 **920 亿美元**
- 2026 CPO 玻璃基板市场约 **120 亿元**
- 2030 Micro Led CPO 超 100 亿美元(CAGR 70%+)
- AIDC"无阻塞直通"架构下光纤消耗 5-10 倍增长

### 5. 国产光互连产业链(15%)
- 光纤(长飞光纤 / 通鼎互联 / 长盈通)
- 光模块(中际旭创 1.6T 50-70% 全球市占 / 新易盛 / 联特科技)
- 光器件(天孚通信 / 东山精密)
- 电芯片(优迅股份 TIA+Driver / 锗硅产能偏紧)
- 上游(长光华芯 CW 激光器 / 三孚股份四氯化硅)
